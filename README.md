# 抖音去水印小程序

## 项目简介

这是一个用于去除抖音视频水印的微信小程序应用。用户可以通过粘贴抖音视频链接来获取无水印的视频，并支持在线播放和下载功能。

## 功能特性

- ✅ 抖音视频链接解析
- ✅ 无水印视频播放
- ✅ 视频下载到本地
- ✅ 链接复制功能
- ✅ 用户友好的界面设计

## 项目结构

```
├── app.js              # 小程序入口文件
├── app.json            # 全局配置文件
├── app.wxss            # 全局样式文件
├── sitemap.json        # 小程序搜索配置
├── pages               # 页面目录
│   ├── index           # 首页（链接输入页面）
│   └── result          # 结果页（解析结果显示页面）
└── utils               # 工具类目录
    ├── api.js          # 抖音解析相关工具
    └── util.js         # 通用工具函数
```

## 使用说明

1. 打开抖音APP，找到想要去水印的视频
2. 点击视频下方的"分享"按钮
3. 选择"复制链接"
4. 在小程序中粘贴复制的链接
5. 点击"立即解析"按钮
6. 在结果页面可以播放或下载无水印视频

## 核心功能实现

### 链接解析流程

1. 用户输入抖音视频链接
2. 程序验证链接有效性
3. 提取视频ID
4. 调用解析接口获取无水印视频地址
5. 展示视频信息和操作选项
